# Docker Image Build Command

**Developement**
```
   docker compose -f compose.dev.yaml build --no-cache && docker compose -f compose.dev.yanl up
```

**Production**
```
   docker compose -f compose.prod.yaml build --no-cache && docker compose -f compose.prod.yanl up
```
